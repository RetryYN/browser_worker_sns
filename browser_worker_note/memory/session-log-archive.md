## Persistent Discoveries (knowledge/ 未反映の重要知見)

### プロジェクト設定
- `.mcp.json` は **リポジトリルート直下** に配置。サブディレクトリ内は Claude Code に認識されない
- Windows: `sys.stdout.reconfigure(encoding='utf-8')` 必須（cp932 で em-dash 等が出力不可）

### ブラウザ操作の安定パターン
- X overlay ブロック: `browser_click` → 失敗 → `browser_evaluate` で `querySelector('[data-testid="..."]').click()` がフォールバック標準
- X textarea: `nativeInputValueSetter` + `input` イベント発火が安定。`slowly: true` は長文でタイムアウト
- X 予約投稿: **時間→日付の順序が必須**（日付先行は過去日時エラー）
- X スレッド追加: 新規作成時のみ。既存予約の編集では「ポストを追加」が disabled
- note beforeunload: `window.onbeforeunload = null` を実行してからページ遷移
- note 画像挿入: **空段落の+メニューから**が鉄則。テキスト段落上で操作するとテキスト分断

### コンテンツ制作
- 画像の英語テキスト禁止。プロンプトに「All text must be in Japanese」必須
- 日本語テキスト描画: 2-4文字が限界。長文は化ける
- X 4画像グリッド: 1枚分割はXのギャップで破綻。独立4回生成が唯一の方法
- 下書き保存フロー: エディタ → 公開に進む → ハッシュタグ設定 → キャンセル（公開せずハッシュタグだけ設定可能）
