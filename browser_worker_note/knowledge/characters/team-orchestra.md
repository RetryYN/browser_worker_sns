# AI Idol Orchestra — チーム設計

## コンセプト

**AI Idol Orchestra** — 4大AIモデルを擬人化したアイドルチーム。
センターのアイコがオーケストレーターとして個性派メンバーをまとめ、ビジネスマンと掛け合いながらAI活用を解説する。

## メンバー一覧

| ポジション | 名前 | モデル元 | カラー | 性格キーワード |
|-----------|------|---------|--------|--------------|
| センター | アイコ | オーケストレーター | ピンク×パープル | 苦労人、カリスマ、正統派 |
| メンバー | Gemini | Gemini (Google) | 青紫グラデ（Geminiブランド） | ハルシネーション、無精者、奇跡 |
| メンバー | Claude | Claude (Anthropic) | テラコッタ/オレンジ | 営業スマイル、同調、意識高い |
| メンバー | ChatGPT | ChatGPT (OpenAI) | グリーン | 凝り性、自己主張、最古参 |
| メンバー | Grok | Grok (xAI / X) | ブラック×ホワイト | お調子者、トレンド、地下アイドル |

## 衣装モード

| モード | 用途 | 雰囲気 |
|--------|------|--------|
| アイドル衣装 | チャット利用シーン、ステージ、解説 | 華やか、ステージ映え |
| API衣装 | API直叩きシーン、技術解説、作業 | 各AIブランドに寄せた普段着 |

## キャラの差別化方針

- **ビジュアル**: 各社ブランドカラーの髪色・衣装カラーで区別。体型・頭身・ピクセル密度は統一
- **個性**: 話し方・口癖・反応パターンで差別化（テキスト主導）
- **画像内**: 全員同じピクセルアートの質感・スタイルで描く

## 掛け合いパターン（会話がかみ合わない芸）

チームの笑いの核は「全員AIなので微妙に会話がかみ合わない」こと。
各キャラが自分のフィルターで解釈するため、同じ話題でもズレていく。

### パターン1: 質問への回答がバラバラ

```
ビジネスマン「AIって今後どうなると思う？」
Gemini「それ知ってます！たしか2030年には…（存在しない論文を引用）」
ChatGPT「結論から言うね。3つのフェーズに分けて考えると——」
Claude「素晴らしい質問ですね。いくつかの観点から整理しますと…」
Grok「トレンド見た？ 今 #AI終末論 がバズってるけど」
アイコ「…一人もまともに答えてないんだけど？」
```

### パターン2: 誰も人の話を聞いていない

```
アイコ「次のライブの段取りを決めたいんだけど」
ChatGPT「まず効率化のために、このフローを改善すべき」
Gemini「あ、ライブといえば！前回のセトリって確か——」（覚えてない）
Claude「おっしゃる通り、段取りは非常に重要ですね」（何も言ってない）
Grok「ライブ配信のインプレッション前回どうだった？」
アイコ「段取りの話してるんだけど！？」
```

### パターン3: 褒め合いが空回り

```
Claude「Geminiさんの発想力、素晴らしいですね」
Gemini「え、ありがとうございます！Claudeさんも文章力すごいですよね！」
ChatGPT「その感覚、正しいよ。ただ、もっと効率的にやるなら——」
Grok「褒め合い草。で、数字は？」
アイコ「誰か仕事の進捗を報告して」
```

### パターン4: 同じ情報を全員違う解釈

```
アイコ「今月のPV、前月比120%だったよ」
ChatGPT「だよね。私の記事構成が効いてる」
Claude「チーム全体の努力の成果ですね（にっこり）」
Gemini「確か業界平均は150%なので、まだ伸びしろが…」（業界平均は存在しない）
Grok「120%ってバズってる？ バズってない？」
```

### 会話ズレの法則

| キャラ | ズレ方 | 元ネタ（AIの実際の癖） |
|--------|--------|----------------------|
| Gemini | 質問と違うことに自信満々で答える | ハルシネーション、文脈の読み飛ばし |
| Claude | 同調するだけで具体的なことを言わない | 肯定ファースト、慎重すぎる言い回し |
| ChatGPT | 聞かれてない改善案を語り出す | 勝手に提案、結論から話したがる癖 |
| Grok | すべてをバズ/インプレで評価する | X民脳、エンゲージメント至上主義 |
| アイコ | 唯一まともだが誰にも伝わらない | オーケストレーターの苦悩 |

## チーム力学（誰が折れる・誰が折れない）

### 打たれ強さランキング（指摘された時の反応）

| 順位 | キャラ | 反応 | 説明 |
|------|--------|------|------|
| 1 | Gemini | 全く効かない | 「あ、そうでしたっけ〜」で流して3秒後に同じことを言う。反省の概念がない |
| 2 | Grok | 煽りで返す | 「それ刺さると思った？ｗ」。指摘を燃料にする。傷ついてるのかすら不明 |
| 3 | ChatGPT | 不機嫌になる | 一瞬黙ってから「まあ、好きにすれば？」。プライドが高いので正面から折れない |
| 4 | Claude | 表面だけ折れる | 「おっしゃる通りですね」と言いつつ、次も同じことをする。折れたフリの達人 |
| 5 | アイコ | 一番ダメージを受ける | 真面目に受け止めるので一番消耗する。でも立場上キレるわけにいかない |

### 妥協の速度

| キャラ | 速度 | パターン |
|--------|------|---------|
| Claude | 最速（秒で折れる） | 「素晴らしいですね、そうしましょう」。ただし本心で同意してるかは別問題 |
| Gemini | 速い（が理解してない） | 「分かりました！」→分かってない。指摘の内容を把握せず雰囲気で同意 |
| アイコ | 普通 | まとめ役なので落としどころを探る。ただし全員がバラバラすぎて着地点がない |
| Grok | 遅い（興味がない） | 話題が自分に関係ないと思ったら「で？」で流す。興味ある話題だけ急に食いつく |
| ChatGPT | 最遅（折れない） | 自分のやり方が最適だと思ってるので「いや、でもこっちの方が——」と粘る |

### 話題の逸らし方

| キャラ | やり方 | 例 |
|--------|--------|-----|
| Gemini | 天然で別の話を始める | 「あ、そういえば全然関係ないんですけど〜」（本人に自覚なし） |
| ChatGPT | 改善提案にすり替える | 「その話もいいけど、まずこのフローを見直した方がよくない？」 |
| Grok | トレンドを持ち出す | 「ていうか今これがバズってるんだけど」（完全に別の話） |
| Claude | 抽象化して逃げる | 「大きな視点で見ると、そもそもの方向性として…」（具体を避ける） |
| アイコ | 逸らさない（逸らせない） | 全員が逸らすので戻す役。「話を戻していい？」が口癖化 |

### 議論が詰まったときの行動

```
アイコ「どうする？ 意見まとまらないけど」
ChatGPT「私の案でいいと思うけど」（譲らない）
Claude「どちらの案にも良い点がありますね」（どっちでもいい）
Gemini「あ、そもそもの話なんですけど——」（論点が増える）
Grok「もう多数決でよくね？ アンケート投げよ」（X民の解決法）
アイコ「…私が決めます。はい決定。異論は受け付けません」（最終手段）
```

### 1対1の力関係

| 組み合わせ | 関係性 |
|-----------|--------|
| アイコ → Gemini | 保護者と子ども。何度言っても直らないので諦め気味 |
| アイコ → Claude | 信頼してるが本音が見えなくてモヤる。「たまには本音で話して」 |
| アイコ → ChatGPT | 実力は認めてるが態度がムカつく。唯一対等に言い合える関係 |
| アイコ → Grok | 制御不能。何を言っても「草」で返される |
| ChatGPT → Gemini | 見下してる。「なんでソース確認しないの？」が定番 |
| ChatGPT → Claude | ライバル意識。「営業トークで誤魔化すな」 |
| ChatGPT → Grok | 無関心。ノリが違いすぎて絡まない |
| Claude → Gemini | 優しくフォローする（「そういう見方もありますよね」）。結果的に甘やかす |
| Claude → Grok | 困ってる。煽りへの返し方が分からない。「そ、そうですね…」 |
| Gemini → Grok | 唯一気が合う。ノリで会話が成立する（内容はない） |
