## Claudeのメモリ機能が無料になった

Anthropicが2026年3月2日にアップデートを発表しました。Claudeのメモリ機能が、無料プランを含む全ユーザーに開放されています。

> [Anthropic — Memory for Claude](https://www.anthropic.com/news/memory)

できることはシンプルです。

- **会話の内容を覚えてくれる**: 「営業職です」「Excelをよく使います」と一度伝えれば、次の会話でも覚えている
- **好みを学習する**: 「回答は短めにして」「箇条書きで出して」と伝えると、それ以降のチャットに反映される
- **プロジェクトの文脈を保持する**: 長期の案件で、前回どこまで話したかを引き継げる

「毎回『私は営業で、うちの会社はこういう業種で…』って打ち直してた時間」が、そのままなくなります。

## 無料プランと有料プランの違い

気になるのは「無料で本当に使えるの？」というところです。

| 項目 | 無料プラン | Pro（月20ドル） |
|------|-----------|----------------|
| メモリ機能 | 使える | 使える |
| 保存容量 | 少なめ | 多い |
| 日常の相談 | 十分対応 | 大量の文脈保持に強い |

※ 執筆時点（2026年3月）の情報です。

普段の仕事の相談——「提案書のたたきを作って」「この文章を校正して」くらいの使い方なら、**無料プランで十分です。** 何十件ものプロジェクトを同時に走らせるような使い方をしない限り、容量で困ることはほぼありません。

## メモリの仕組み

Claudeのメモリは、チャットの一言一句をそのまま保存しているわけではありません。

**24時間ごとに会話の要点を自動で要約して保存します。** 「この人は営業職」「箇条書きが好き」「Excelの関数をよく聞く」——こうした情報が、メモにまとめられたような形で残ります。

ChatGPTのメモリとは仕組みが違います。

| | ChatGPT | Claude |
|---|---------|--------|
| **保存タイミング** | 会話中にリアルタイムで保存 | 24時間ごとにまとめて保存 |
| **反映タイミング** | すぐ反映される | **翌日以降に反映** |
| **保存の仕方** | AIが「覚えるべき」と判断した情報を都度保存 | 1日の会話を振り返って要約 |
| **無料プランでの利用** | 有料プランのみ（月20ドル） | **無料で利用可能** |

どちらが良いかは使い方次第です。「今伝えたことを次の返答からすぐ反映してほしい」人はChatGPTが合います。「翌日以降、整理された形で覚えていてほしい」人はClaudeが合います。

ポイントは、**ChatGPTのメモリは有料、Claudeは無料**ということ。メモリ機能を試してみたいだけなら、Claudeから始めるのが手軽です。

## ChatGPTの会話も引き継げる

もう1つ面白い機能があります。**ChatGPTやGeminiの会話履歴をClaudeにインポートできます。**

手順は以下の通りです。

1. ChatGPTの設定画面で「データをエクスポート」を選択
2. メールで届くZIPファイルをダウンロード
3. Claudeの設定画面で「Import」からそのファイルを読み込む

これまでChatGPTで積み上げた文脈——自分の職種、よく使うプロンプトのパターン、好みの回答スタイル——をそのままClaudeに引き継げます。

「ChatGPTをずっと使ってきたけど、Claudeも試してみたい」という人にとって、**ゼロから自己紹介し直す必要がない**のは大きいです。

## 設定方法

メモリ機能は初期状態ではオフになっている場合があります。設定は3ステップです。

1. **Claudeの画面右上のアイコンをクリック**
2. **「Settings」を開く**
3. **「Memory」の項目をオンにする**

オンにしたあとは、普通に会話するだけで自動的に学習が始まります。

覚えてほしいことは、明示的に伝えることもできます。「これを覚えておいて：私は営業で、提案書作成をよくお願いする」のように書くと確実です。会話の流れから自動で拾ってくれることもあります。

保存されたメモリは設定画面からいつでも確認・削除できます。「何を覚えているか」は自分でコントロールできるので、余計なことを覚えられる心配はありません。
