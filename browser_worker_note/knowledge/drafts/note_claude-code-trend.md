## 事例1: データ分析の世界大会で3,803チーム中5位。コードを書いたのはAI

Kaggleというデータ分析の世界大会があります。企業や大学のデータサイエンティストがチームを組んで、予測精度を競うコンペです。

2026年2月、あるコンペで**Claude Codeにコードを全て書かせて5位に入った**事例が公開されました。3,803チーム中の5位、上位約0.1%です。

注目すべきは、この人が実際にやったことです。

- **コードは1行も書いていない。** プログラムの中身はすべてClaude Codeが生成した
- **やったのは「方針決め」と「判断」。** 「このデータをこう分析して」「この手法を試して」と指示を出し、結果を見て次の指示を出した
- **AIに何十パターンも試させた。** 1つの手法に固執せず、大量に試行して良い結果が出たものを採用した

つまり、プログラミングの技術力ではなく、**「何を試すか」を決める判断力**で上位に食い込んだということです。

> 参考: genda_jp「Claude Codeを使ってKaggleコンペで金メダルを獲得した話」（Zenn、2026年2月）

## 事例2: スタッフ0人の税理士事務所が60社を担当している

もう1つの事例は、まったく違う業界です。

ある税理士事務所では、**事務スタッフを雇わずに1人で60社のクライアントを担当**しています。通常、60社を回すには数人のスタッフが必要な規模です。

この事務所でClaude Codeが担っている作業はこちらです。

- **書類作成の下書き**: 税務申告書や届出書の下書きをAIに作らせる
- **メールの下書き**: クライアントへの定型連絡をAIが作成し、税理士が確認して送信
- **データ整理**: 領収書や請求書のデータ入力をAIに任せる

税理士本人がやっているのは、最終確認と判断が必要な部分だけです。税法の解釈や、クライアントへの重要な助言は人間がやる。それ以外の「手を動かす作業」をAIに渡しています。

> 参考:「Claude Codeで無人税理士事務所を実現した話」（Cenleaf blog、2026年3月）

## 2つの事例の共通点

業界もスキルもまったく違う2人ですが、共通していることがあります。

**「AIに何をやらせるか」を決められる専門知識を持っていること。**

- Kaggleの事例では、データ分析の知識がなければ「この手法を試して」という指示が出せない
- 税理士の事例では、税務の知識がなければ、AIの出力が正しいかどうか判断できない

AIがコードを書ける、書類を作れる。それは事実です。でも、**指示と判断ができなければ使えません。** 逆に言えば、自分の仕事の専門知識さえあれば、プログラミングができなくてもAIに実作業を任せられる時代になっています。

## 「丸投げ」の正体は「指示と検収」

「AIに丸投げ」と聞くと、ボタン1つで全部終わるように聞こえます。でも実態はもっと地味です。

**管理職が部下に仕事を振るのと同じ構造です。**

- **丸投げできるのは「作業」の部分だけ。** コードを書く、文書を整形する、データを入力する
- **人間がやるのは「設計」と「検収」。** 何をやるか決めて、結果が正しいか確認する

部下（AI）がどれだけ優秀でも、何を任せるか決めて、成果物をチェックするのは上司（人間）の仕事です。

これをあなたの仕事に当てはめると、こうなります。

| あなたがやること | AIに渡せること |
|----------------|--------------|
| 企画の方針を決める | リサーチ、情報収集、下調べ |
| 報告書の構成を考える | 文章の下書き、データ整理 |
| クライアントへの提案内容を判断する | 提案資料のたたき台作成 |
| メールの要点を決める | 定型文の下書き |

2つの事例が示しているのは、AIが「作業者」として実用レベルに達しているということです。ただし、**指示書を書いて成果物を検収する力**は、これまで以上に求められます。

## 今日からできること: 「判断不要な30分」を見つける

難しく考える必要はありません。第一歩は、**自分の仕事の中で「時間はかかるけど判断は要らない作業」を1つ見つける**ことです。

例えばこんな作業です。

- 会議の議事録を整理する
- 定型メールの下書きを作る
- データを決まったフォーマットに転記する
- リサーチ結果を一覧表にまとめる

こういった作業を、まず1つだけChatGPT（無料版でOK）に任せてみてください。「この議事録を箇条書きにまとめて」「このメールの返信の下書きを作って」と日本語で伝えるだけです。

指示の出し方が分かれば、2つ目、3つ目と任せる範囲を広げられます。**今日の仕事で30分かかっている作業、1つだけ思い浮かべてみてください。**
