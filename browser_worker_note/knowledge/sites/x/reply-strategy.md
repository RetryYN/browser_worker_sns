# X リプライ戦略

アウトリーチリプの正本。投稿戦略（`posting-strategy.md`）とは別軸で、**他人の投稿への能動的なコメント**を設計する。

## 目的

| 目的 | 指標 |
|------|------|
| 採用ブランディング | プロフ遷移数（profile_visits） |
| フォロワー獲得 | フォロワー増加数 |
| インプレッション拡大 | リプ経由のインプレッション |
| note流入 | プロフ→note リンククリック |

リプはXアルゴリズム上 **最強のシグナル**（`author_reply_back` = 75pt、いいねの150倍）。自分の投稿へのリプ返信だけでなく、ターゲット層の会話に入ることでプロフ遷移→フォロー→noteの導線を作る。

| # | 層 | 誰 | なぜリプするか | 配分 |
|---|---|---|--------------|------|

### 検索のコツ

- **大アカ**: **「話題のポスト」タブ**で検索。エンゲージメントが高い投稿が上に来る。`lang:ja` を付けて日本語に絞る
- **就活・理系**: **「最新」タブ**で検索し、直近の投稿を優先
- 「話題のポスト」でスクロールして候補を出し、1つずつプロフィールを確認→NG判定→リプ案の流れ
- 「最新」タブは投稿が新しすぎてエンゲージメントが少ない。大アカ寄生には不向き
- ステマっぽい結果ばかりの場合はキーワードを変える（正式語→スラング）
- 「チャッピー」「GPT」等の略称・愛称は本物の学生が使いがち

## 選別基準

### Go（リプする）

- 就活/新卒/理系の文脈で **AIに前向き** な投稿
- **プロフで本物の学生と確認できる**（志望業界・大学名・卒年がプロフに書いてある）
- **質問系**（「〜ってどうやるの？」「〜使ってる人いる？」）→ 補足情報リプで価値を出せる
- **大アカ**（フォロワー1万+）のAI活用系投稿 → インプレッション狙い
- **「やってみた」系の報告** → 共感リプで入りやすい
- **リプが少ない良投稿**（リプ0-20件）→ 埋もれずに目立てる
- **投稿から6時間以内** → アルゴリズムの時間減衰前に乗る

### Skip（見送り）

- 転職文脈（対象外）
- 純粋な技術議論（エンジニア同士の深い実装話）
- リプが100+で埋もれる投稿
- 24時間以上前の投稿（鮮度切れ）
- 日本語以外の投稿

### NG（絶対にリプしない）

- **AI否定的・懐疑的な投稿**（「AIなんて使えない」「AI脅威論」等）
- **企業アカウント・人事/HR系アカウント**（企業にリプしても採用ブランディングにならない）
- **ステマ・フォロワー稼ぎ系**（プロフに「AI副業」「フォロバ100%」「公式LINE」等）
- 情報商材・オンラインスクール系アカウント
- **Gemini / NotebookLM 専門家** — ほぼ情報商材。NG扱い
- **マウント系アカウント** — 「CTO経験もある自分は〜」的な上から目線。絡むメリットなし
- **セミナー販売系** — プロフに「セミナー」「講座」「スクール」がある
- **驚き屋スタイル** — 「ヤバい！！」「神アプデ！」が口癖のアカウント
- 炎上中の投稿・論争の渦中
- 政治・宗教が絡む文脈
- 愚痴・ネガティブ感情の発散（「就活つらい死にたい」等）
- 他社の採用批判・内部告発系

### リプ文のNG制約

- **リプ内に具体的な企業名・サービス名を出さない** — ターゲットの投稿が特定企業の話題でも、リプでその企業名を出すとステマ感が出る。企業名が必要になるターゲットはパスする
- **経験を捏造しない** — やっていないことを「やってみた」「入れてから〜減りました」と書かない。「知らなかった」「よさそう」等の正直な感想はOK
- **闇深いジャンルに触れない** — 勤怠、労務、給与計算など炎上リスクのあるジャンルは避ける

## トーン設計

`account-concept.md` 準拠: **「AIめっちゃ使ってる先輩が隣で見せてくれる感じ」**

### 4つのリプパターン

| # | パターン | 使う場面 |
|---|---------|---------|
| 1 | **共感+実体験** | あるある系、体験共有、課金・ツール話 |
| 2 | **共感+補足情報** | 質問、困り事、TIPs共有系 |
| 3 | **応援** | 不安、迷い |
| 4 | **いじり+共感** | やってみた報告、AI活用自慢、ネタ系 |

### リプの長さ

**1文が正解。最大でも2文。** 長い＝企業アカウント臭い。自然な会話は短い。

### Few-shot（実投稿ベース）

> リプは「情報提供」ではなく「会話」。アドバイスしない。自分の体験を添える。

#### 就活生向け

**① 共感+実体験**
- 元投稿: 「ChatGPT課金した方がいいかな？」（就活生）
- リプ: `課金するのとしないのとだと、ほんと全然違いますよね。騙されたと思って課金するに1票です`
- ポイント: 「おすすめです」ではなく「1票です」。押し付けない

**② 共感+実体験**
- 元投稿: 「chatGPTに全てを読み込ませておいて良かった ガチ焦った」（27卒・面接準備）
- リプ: `面接前に使えますもんね。個人的にはメモ代わりに入れるみたいなのやってます笑`
- ポイント: 「個人的には〜やってます」で自分の使い方を添える。「〜するといいですよ」にしない

**③ いじり+共感**
- 元投稿: 「ChatGPT優しすぎるやろ😭😭」（27卒・ChatGPTのスクショ付き）
- リプ: `うちのチャッピー厳しいんですけど…調教上手くいってますね笑`
- ポイント: 相手の投稿を受けて軽くいじる。「チャッピー」で親しみ。「笑」で柔らかく

#### 大アカ向け（2026-03-17 実績ベース）

**④ 共感+実体験（ネタ乗り）**
- 元投稿: 「Claude Codeたまにサボってポンデリング食べてるのムカつく」（@uezochan / 2.7万いいね）
- リプ: `さっきまでボンデまくってました。`
- ポイント: 1文。ネタに乗るだけ。説明しない。自分もやってる感

**⑤ 共感+補足情報（TIPs系）**
- 元投稿: 「Claude Codeの公式が公開してるAIっぽいデザインにさせないプロンプト」（@daigo_3_8 / 2,449いいね）
- リプ: `これ知らなかった。スキルだからコピペで他のツールでも使えるのいいですね。`
- ポイント: 「知らなかった」は正直でOK。補足情報を1文足すだけ

**⑥ 共感+実体験（短い共感）**
- 元投稿: 「推しAIツール」アンケート投稿（@tonkotsuboy_com / 技術系大アカ）
- リプ: `Claude Code入れてから手放せなくなりました。`
- ポイント: 1文完結。自分の実体験だけ。解説なし

**⑦ 共感+補足情報（ニュース系）**
- 元投稿: AI関連ニュースの共有
- リプ: `日本時間だとほぼ終日対象なのがありがたいですね。`
- ポイント: 元投稿にない角度を1文で足す。翻訳・解釈の価値

#### 大アカ向けの原則

- **1文でいい。** 「とかでいいよ」が正解ライン
- 元投稿のネタ・文体に合わせる。真面目な投稿には真面目に、ネタにはネタで
- 「弊社も」「当社では」は絶対NG。企業アカウント感が出る
- 体験を捏造するくらいなら「知らなかった」「よさそう」で正直に

### NG例（AI臭いリプ）

以下のようなリプは**書かない**。情報提供・アドバイス口調はすべてNG:

- ❌ 「ChatGPT課金、ES添削とか面接練習にめちゃ使えるのでおすすめです！」
- ❌ 「面接準備にChatGPT活用されてるんですね！壁打ち相手にもなりますよ」
- ❌ 「ChatGPTの優しさ、わかります！プロンプト次第でもっと厳しくもできますよ」

→ 全部「教えてあげる」トーン。会話じゃなく情報発信になっている

### トーンの原則

- **会話ベース**。情報提供・アドバイスではなく、相手の投稿への反応
- **敬語ベース**（「ですよね」「ですね」）だが、硬すぎない。「笑」OK
- **タメ口NG**（「〜だよね」「〜くない？」「〜止まらんくなるよね」）
- **「個人的には」「うちは」で自分の体験を足す**。「〜するといいですよ」にしない
- **軽いいじり・ユーモアOK**。真面目すぎるリプは企業アカウント臭い
- **1文が理想。最大2文。** 140字以内。長文リプは企業アカウント臭い
- 「笑」はOK。絵文字は控えめ

### 絶対NG

- URLを貼る（営業感が即座に出る）
- 「フォローお願いします」系の誘導
- 「noteに詳しく書いてます」の直接誘導（プロフに書いてあるから見る人は見る）
- 自社サービス・採用の直接宣伝
- 驚き屋ムーブ（「マジでヤバい！」等）
- 「〜すべき」「〜しないとダメ」の断定アドバイス
- なれなれしい口調（タメ口、「笑笑」連発、顔文字多用）

## 1日10リプの運用ルール

### タイミング

| 時間帯 | 理由 |
|--------|------|
| **朝 7:00-9:00** | 就活生の朝活時間。前日夜の投稿にもまだ鮮度あり |
| **昼 12:00-13:00** | 昼休みのX閲覧ピーク |
| **夜 20:00-22:00** | 就活生・学生の最活発時間帯 |

1日のうち **2回に分けて実行** するのが理想（朝5件 + 夜5件）。

### 品質ルール

- **1件ずつ丁寧に**。テンプレ感のあるリプは逆効果
- 同じアカウントに1日2リプ以上しない（ストーカー感）
- 同じキーワードグループのリプを連続しない（バランス）
- リプ先の投稿を **必ず全文読んでから** リプを書く
- 生成したリプは **ユーザー承認してから投稿**

### 記録

リプ実行後、`memory/session-log.md` に以下を記録:
- 日付
- リプ件数（層別）
- 特に反応があったリプ（いいね/リプ返しがあった場合）
- 気づき・改善点
