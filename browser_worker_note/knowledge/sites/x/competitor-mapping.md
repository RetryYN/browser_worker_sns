## 勝ちパターンと空白地帯

### 勝ちパターン（高エンゲージメント）

1. **C4×P6×T5（体験×Before/After×実践）** — 「やってみたらこう変わった」
   → kandmybike, AI_EC_Hacker, nezumi_uranai が代表。画面スクショ必須
2. **C4×P9×T5（体験×ケーススタディ×実践）** — 「この業務をこう自動化した全記録」
   → kandmybike の最強パターン。note長文向き

   → masahirochaen, Takuma_Uemura_ のX量産型

## NG層との構造的な違い

| 要素 | NG層の構造 | ベンチマーク層の構造 |
|------|----------|------------------|
| C（内容） | C2 ノウハウが多いが**自分で試していない** | C4 体験が主軸。**実際にやった結果**がある |
| P（構成） | P5 リストに偏る（○選、まとめ） | P6 B/A, P9 ケーススタディ, P2 ステップが分散 |
| T（加工） | T1 整理だけ（集めて並べただけ） | T5 実践 + T3 解釈（やった上で意味づけ） |
| 深さ | 広く浅い（プロンプト200個） | 狭く深い（1業務を徹底解剖） |
| 証拠 | なし or 他人の成果スクショ | 自分の画面スクショ、実行ログ、数値 |
| 読後感 | 「へー」で終わる | 「自分もやってみよう」になる |

### NG判定チェックリスト（コンテンツ制作時）

制作した記事が以下に該当したらNG層に堕ちている:
- [ ] 自分で実行していない情報をまとめているだけ
- [ ] ○選リストで項目数が価値の中心になっている
- [ ] 具体的な画面スクショ・数値データがない
- [ ] 読者が読んだ後に再現できるステップがない
- [ ] 「すごい」「革命」「損する」等の感情操作ワードに頼っている
