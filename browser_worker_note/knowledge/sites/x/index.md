### 予約投稿設定（/compose/post/schedule）

「ポストを予約」クリックで表示。**プレミアム不要で利用可能**。

| 要素 | 型 | 備考 |
|------|-----|------|
| 月 | combobox | 1月〜12月 |
| 日 | combobox | 1〜31 |
| 年 | combobox | 2026〜2028 |
| 時 | combobox | 0〜23（24時間表記） |
| 分 | combobox | 00〜59（1分刻み） |
| Calendarボタン | button | カレンダーUI |
| タイムゾーン | 表示のみ | 日本標準時 |
| 確認する | button | 予約設定を確定してポスト作成に戻る |
| 予約投稿ポスト | button | 一覧表示（フッター） |

操作手順:
```
1. select_option "月" → 月を選択
2. select_option "日" → 日を選択
3. select_option "年" → 年を選択
4. select_option "時" → 時を選択
5. select_option "分" → 分を選択
6. click "確認する" → ポスト作成画面に戻る
7. ポスト本文を入力
8. click "予約設定する" → 予約完了
```

## 注意事項

- **ログインフロー**: 2段階（メール入力画面 → パスワード入力画面）。各画面でのref値はセッションごとに変わる
- **初回ロード**: SPAのため5秒待機推奨
- **ポスト文字数制限**: 140文字（プレミアムなら拡張可能）
- **beforeunloadなし**: noteと違い、X側では離脱時のダイアログなし
- **予約投稿**: プレミアム不要。comboboxで日時指定
