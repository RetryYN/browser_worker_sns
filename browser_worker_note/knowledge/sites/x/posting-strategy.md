# X 投稿戦略

運用スケジュール・投稿パターン・ネタ設計の正本。
画像仕様は `image-regulation.md`、サイト操作は `index.md` を参照。

## 運用サイクル

- **投稿頻度**: 毎日4件（あさいち・お昼・帰宅時間・深夜の4枠）
- **起動タイミング**: 毎日（まとめて予約投稿してもOK）
- **やること**: 当日〜翌日分の予約投稿を作成 + レポート作成
- **リプ返信**: 投稿後1時間以内に返信（75pt/回、生存率に直結）
- **アウトリーチリプ**: 1日10リプ（就活生・理系学生・大アカ向け。詳細は `reply-strategy.md`）

## 投稿パターン（10型）

### S級: リプライ誘発 × 画像（最優先）

#### 1. どっち派？対決
- **フォーマット**: ポール + キャラ対決画像2枚
- **生存理由**: ポール=即タップ、画像=2倍、「俺はこっち派」リプ自然発生
- **難易度**: 初心者〜中級
- **配置**: 月曜夜
- **ネタ例**:
  - 「文章の要約、Claude派？ChatGPT派？」
  - 「アイデア出しに使うなら Gemini？ChatGPT？」
  - 「メール文面の添削、どっちに頼む？」

#### 2. 参加型お題
- **フォーマット**: お題テキスト + キャラ例示画像1枚 + リプ誘導
- **生存理由**: リプ誘発が最大目的。全リプに返信で75pt×n
- **難易度**: 初心者
- **配置**: 月曜夜
- **ネタ例**:
  - 「あなたの #AIあるある 教えて」
  - 「AIに言われてイラッとした一言は？」
  - 「AI使ってて一番助かった場面を教えて」

#### 3. AI選手権クイズ
- **フォーマット**: クイズ画像1枚（`quiz-choice`）+ ポール
- **生存理由**: ポール即タップ、正解は翌日引用RTで2日分の滞在
- **難易度**: 初心者
- **配置**: 月曜夜
- **ネタ例**:
  - 「この回答をしたのはどのAI？」
  - 「このプロンプトへの返答、一番マシなのは？」
  - 「この"素晴らしいですね"を言ったのは誰？（全員Claude）」

---

### A級: 滞在時間 × 画像（安定主力）

#### 4. Xグリッド4コマ
- **フォーマット**: 画像4枚（1枚=1コマ、起承転結）
- **生存理由**: 4枚見る=滞在時間×4。オチが気になって止まる。ブックマーク率高
- **難易度**: 初心者
- **配置**: 水曜昼
- **ネタ例**:
  - 「AIに議事録を頼んだら」→各キャラが的外れな対応→アイコ絶望
  - 「新人にAIの使い方を教える会議」→全員が自分推し→まとまらない
  - 「AIにプレゼン資料を頼んだ結果」→ChatGPTが勝手にフル改善

#### 5. プロンプト before/after
- **フォーマット**: Before画像 + After画像（2枚）
- **生存理由**: 実用的=ブックマーク率高。「これやってた…」リプ誘発
- **難易度**: 初心者〜中級
- **配置**: 水曜昼
- **ネタ例**:
  - ❌「記事書いて」→ ✅「○○について、読者は△△、800字で」
  - ❌「翻訳して」→ ✅「カジュアルな日本語で、専門用語は英語のまま」
  - ❌「要約して」→ ✅「3行で、結論→根拠→補足の順で」

#### 6. 使い分けチャート図解
- **フォーマット**: 情報密度の高い図解画像1枚
- **生存理由**: 滞在時間最大化。ブックマーク率最強。note記事導線
- **難易度**: 中級
- **配置**: 水曜昼
- **ネタ例**:
  - 「この作業→このAI」フローチャート
  - 「AI4種の得意・不得意マトリクス」
  - 「プロンプトの型：指示・文脈・制約・出力形式」

---

### B級: 拡散 × キャラ力（認知拡大）

#### 7. あるある掛け合い
- **フォーマット**: 短テキスト（140字以内）+ キャラ集合画像1枚
- **生存理由**: 短い=RT閾値低い。「あるあるｗ」共感拡散。新規フォロワー入口
- **難易度**: 初心者
- **配置**: 月曜 or 金曜（S級の代打）
- **ネタ例**:
  - 「全員に"何か質問ある?"と聞かれた結果」→4者4様のズレ
  - 「締め切り前日のAIたち」→ChatGPT改善始める/Claude"計画的にやりましょう"
  - 「AIに"短くして"と頼んだ結果」→Claude"かしこまりました（長文）"

#### 8. 一言毒舌レビュー
- **フォーマット**: キャラ4人の一言を並べた画像1枚
- **生存理由**: 短い+毒舌=RT誘発。「○○が的確すぎるｗ」リプ
- **難易度**: 中級
- **配置**: 金曜夜（ニュースに応じて）
- **ネタ例**:
  - 「GPT-5発表を各AIが一言レビュー」
  - 「新しいAIツールXXを使ってみた感想」
  - 「今週のAIニュースを一言で」

---

### C級: 深掘り × 信頼構築（ガチ回）

#### 9. ガチ解説スレッド
- **フォーマット**: 4〜6ツイート + 各ツイートに画像
- **生存理由**: スレッド=3倍。各ツイートにリプ分散=合計スコア高。ブックマーク率高
- **難易度**: 上級
- **配置**: 金曜夜（月1〜2）
- **ネタ例**:
  - 「Claude Codeで1週間開発してみた：自動化できたこと・できなかったこと」
  - 「MCPサーバーを実務で使ってみた結果」
  - 「AIエージェント、結局何が手動なのか問題」

#### 10. トレンド速報キャラ実況
- **フォーマット**: ニュース要約テキスト + キャラ反応画像1枚
- **生存理由**: 速報=初動エンゲージ確保（半減期52分に間に合う）。トレンド便乗
- **難易度**: 中級
- **配置**: 不定期（トレンド発生時、月2〜4）
- **ネタ例**:
  - 「○○社が新モデル発表」→Grok速報→他キャラツッコミ
  - 「AI規制の新方針が話題に」→各キャラのフィルターで解釈
  - 「話題のAIツールが登場」→全員が自分の優位性を主張

## 毎日4投稿の配置ルール

```

├─ 画像必須（diagram推奨。スコア2倍を捨てない）
│
12:00 お昼（滞在時間 = A級優先）
├─ 図解・4コマ・before/after など画像メイン

├─ どっち派？対決、参加型お題、クイズ
├─ リプが最も付きやすい時間帯。全リプ返信で75pt×n
```

## 全投稿共通ルール

1. **画像は必ず付ける**（スコア2倍を捨てない）
2. **リプには必ず返す**（75pt/回）
3. **投稿後1時間はリプ監視・返信に充てる**（半減期52分）
4. **140字以内を基本とする**（短い方がRT閾値が低い）
5. **ハッシュタグは最大1個**（2026年アルゴリズムではハッシュタグ乱立はマイナス）
6. **URLリンクは避ける**（外部リンクはアルゴリズムで不利。note導線はプロフ誘導で）
7. **トレンド速報は通常スケジュールに割り込み可**（半減期を逃さない）
8. **フォロー誘導は週1回まで**（押し売り感を出さない。自然な文脈で）

### NGワードチェックリスト

投稿前に以下が含まれていないことを確認する。

| NG | 理由 |
|----|------|
| 衝撃、ヤバい、神ツール、革命 | 驚き屋ムーブ |
| 正直に言うと、ぶっちゃけ、本音 | AI頻出メタ前置き。逆に嘘っぽくなる |
| たった○日で、劇的に | 煽り表現 |
| キャラ名をフック位置に使用 | 内輪感が出る |

## note連動パターン

note記事を公開した際のX投稿ルール。

### 基本形: プロフ誘導

```
[記事の核心を140字で要約]
詳細はプロフのnoteへ
```

- URLを本文に入れない（アルゴリズム不利を回避）
- プロフィールにnoteリンクがある前提
- スレッドでURL追加も非推奨（予約投稿でスレッド追加不可）

### note→X転用の切り口

| note記事の要素 | X投稿の切り口 | 推奨パターンID |
|---------------|-------------|--------------|
| 記事の結論 | 結論だけ抜き出し+「詳細はnoteへ」 | `trend-sokuhou` |
| 比較表・図解 | 図解画像をそのまま転用 | `tsukaiwake-chart` |
| 数値・データ | 数値フックで興味喚起 | `prompt-ba` |
| 体験談の核心 | あるある化して共感誘発 | `aruaru-kakeai` |
| 手順の要約 | 3行にまとめて画像付き | `tsukaiwake-chart` |

### 確定アクション

1. **参加型お題（S級）を一時凍結**: フォロワー500人達成まで投入しない。S級枠はnote連動 or trend-sokuhouで代替
2. **朝枠はnote連動 or trend-sokuhouに固定**: 朝tipsは完全廃止。データが明確
3. **昼枠×図解は週3回以上を維持**: 最強の組み合わせ

### 次の実験（3件）

1. **note連動を朝枠に投入**: 朝×note連動の組み合わせを検証。note連動はどの時間帯でも強い仮説
2. **夜枠にtrend-sokuhouを投入**: 夜×速報の相性を検証（速報は時間帯不問のはず）
3. **screenshotタイプの効果測定**: 実スクショ画像が生成画像より反応良い仮説（Slot 12で初投入）

### 検証指標

- 実験1: 朝×note連動のavg_impが30以上になるか
- 実験2: 夜×trend-sokuhouのavg_impが50以上になるか
- 実験3: screenshot画像のimp vs 同テーマdiagram画像のimp
- 計測タイミング: 3/21以降（今回バッチ12件の計測後）
