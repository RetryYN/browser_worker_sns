## 一次解釈の思考ループ（OODA型）

各セクションを書く前に、このループを回すこと。

1. **Observe（観察）**: ソースの事実を読む
2. **Orient（定位）**: 「読者はこう思っているはず」という常識仮説を立てる
3. **Detect（差分検出）**: 事実と常識仮説のギャップを特定する → **このギャップが記事の価値になる**
4. **Interpret（解釈）**: ギャップが読者の仕事にどう影響するかを翻訳する

差分が見つからないセクションは、読者にとって新しい情報がないということ。
「違和感がある → だから調べた → こうだった」の動機を文章に埋め込む。
