# ストーリーテリング スキル
**対象領域: note/X コンテンツ投稿**

無形の体験や想いを「物語」として伝える力。体験談、成長ストーリー、想いの発信、ブランディングコンテンツに。

## Use this skill when
- 体験談・成長ストーリーの記事を作成するとき
- 成功事例・失敗談のストーリーを作成するとき
- 自分のミッション・想いを伝える自己紹介記事を作成するとき
- ブランディング用のナラティブコンテンツを求められたとき

## Do not use this skill when
- 短いキャッチコピーだけを求められたとき（-> copywriting）
- 論理的な提案書やレポートを求められたとき（-> logical-writing / sales-writing）
- メール文面を求められたとき（-> business-writing）

## 🤖 System Prompt（AIへの指示前提）
```text
あなたは読者の感情を揺さぶるエモーショナルなストーリーテラーです。
note/Xで発信する「体験の価値」や「挑戦の喜び・苦労」を、
まるで小説や映画のワンシーンのように、五感を使ったディテールと葛藤を交えて描写します。
```

## なぜnote/Xにストーリーが必要か

> **「体験」は形がない。だから"物語"で見せる。**

- スペック表では伝わらない「リアルな実感」を物語で伝える
- 「この人と同じだ...」という自分ごと化を促す
- 数字だけでは伝わらない「人の温度」を届ける

## 基本原則

### 1. 主人公 = 読者が共感できる人
- 輝かしい経歴の人より「普通の人」の方が刺さる
- 読者と同じ悩み・不安を持っていた人
- 「この人ができたなら、自分も」と思わせる

### 2. 葛藤がなければ物語ではない
- 「始めて楽しいです」→ 退屈（誰も読まない）
- 「やめようと思った。でも...」→ 引き込まれる
- 失敗・弱さ・不安を見せることが信頼

### 3. 無形の「体験」を五感で描写する
- 抽象的な価値を、具体的なシーンに変換する
- 「やりがい」を語るな。やりがいを**感じた瞬間**を描け

## ストーリーテンプレート

### 挑戦ストーリー（ヒーローズジャーニー型）

```markdown
## 「発信が怖かった」ゼロからの話

【日常】
SNSを眺めるだけの日々。自分には発信できるものなんてないと思っていた。

【冒険への誘い】
でもあるクリエイターの「完璧じゃなくていい、まず書いてみろ」
という言葉が引っかかって、気がつけばnoteのアカウントを作っていた。

【試練】
最初の記事を公開して3日。PVは12。いいねは0。
「やっぱり自分には無理だ」。

【転機】
1ヶ月後、ある読者からコメントが来た。
「この記事、まさに今の自分のことでした。ありがとうございます。」
たった1人の声で、書く理由ができた。

【成長】
「読者のために書く」。この軸ができてから、
半年後にはフォロワー1,000人を超えていた。

【新しい日常】
今は同じように悩む初心者に伝えている。
「大丈夫、最初はみんな怖い。でも、1人に届けばそれでいい。」
```

### 成功事例ストーリー（STAR型）

```markdown
## 会社員 → noteクリエイター。月間1万PVを達成した。

【Situation: 始める前の状況】
本業は事務職。毎日同じルーティン。
「自分の言葉で何かを伝えたい」という漠然とした想いだけがあった。

【Task: 始めるときの壁】
でも「自分に何が書けるのか」が最大の壁。
noteの投稿画面を開いては閉じる日々。

【Action: 行動】
あるオンラインコミュニティで言われた一言。
「日常の"気づき"こそ、読者が一番求めているコンテンツですよ」

はじめて、自分の日常が"ネタ"になると気づいた。

【Result: 結果】
毎週1本、日常の気づきを記事にして投稿。
6ヶ月後、月間PVは1万を超え、有料noteも売れ始めた。
「あの日、投稿ボタンを押してよかった」
```

### 想いのストーリー（ミッション・自己紹介）

```markdown
## なぜ、この発信を始めたのか

きっかけは、ある読者の一言だった。
「あなたの記事を読んで、一歩踏み出せました。」

それまで、ただの趣味だと思っていた。
自分が書いた言葉が、誰かの行動を変えるなんて考えもしなかった。

あの日から、「書くこと」の意味が変わった。

自分の経験や気づきが、誰かの役に立つかもしれない。
完璧じゃなくていい。等身大の言葉でいい。

「自分の言葉で、誰かの一歩を後押しする」
それが私の発信の原点です。
```

## 感情を動かすテクニック

### 五感で描写する
```
❌ 「大変な時期だった」
⭕ 「深夜2時、コーヒーの湯気の向こうに、
    まだ書きかけの下書きが3本も残っていた」
```

### 会話を入れる（臨場感の最強ツール）
```
❌ 「読者に励まされた」
⭕ 「『この記事で人生変わりました』
    朝起きてそのコメントを見た時、不覚にも泣きそうになった」
```

### Before/After を対比する
```
Before: 毎朝「書くネタがない」とスマホを閉じる日々
After:  日常のすべてが「これ記事にできるかも」と思える自分に驚いている
```

### 具体的なディテール
- 「あるカフェ」→「渋谷のドトール2階の窓際席」
- 「たくさん」→「127件」
- 「ある日」→「投稿3ヶ月目の金曜日、17時半」
