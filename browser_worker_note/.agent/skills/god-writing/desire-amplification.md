# 欲求増幅（note / X 特化）

読者の「欲しい」「やりたい」を強め、行動への動機を高めるスキル。
NOT Act（行動しない）突破の武器。

---

## 欲求の原理

- 人は欲求を満たすために行動する
- 欲求が強いほど行動の可能性が高い
- 方向は2つ: **快を求める**（得たい）と **苦を避ける**（失いたくない）

### ミサキ層の欲求トリガー

| トリガー | ミサキ層での具体例 |
|---------|-------------------|
| 時間を節約したい | Excelの関数調べる時間をなくしたい |
| 楽をしたい | 毎回同じ作業をAIに任せたい |
| 認められたい | 「AIに詳しい人」と思われたい |
| 不安を解消したい | 「AIに仕事を奪われる」が怖い |
| 成長したい | AIを使いこなせる側になりたい |
| 取り残されたくない | 周りが使い始めてるのに自分だけ… |

**このアカウントで使わないトリガー:**
- お金を稼ぎたい（情報商材っぽくなる）
- モテたい（テーマと無関係）
- 緊急性の捏造（驚き屋ムーブ）

## テクニック

### 1. コントラスト（Before/After）

現状の痛みと理想の姿の落差で欲求を強める。

**このアカウントでの使い方:**
```
Before: 「VLOOKUP ググる → コピペ → エラー → またググる → 30分消える」
After: 「日本語で指示 → 30秒で完成」
```

落差が大きいほど効く。ただし**誇張しない**（信頼構築優先）。

### 2. フューチャーペーシング

理想の未来を具体的に想像させる。

**このアカウントでの使い方:**
```
NG: 「想像してください。AIを使いこなす未来を。」（セミナー臭い）
OK: 「毎週月曜のレポート作成、30分で終わるようになります。」
```

ミサキ層向けでは**日常業務の具体シーン**で描写する。抽象的な「輝く未来」は響かない。

### 3. 感情のエスカレーション

段階的に感情を高めていく。いきなりクライマックスにしない。

```
1. 共感: 「Excelの関数、毎回ググってませんか」
2. 認識: 「実はその時間、月に5時間以上あります」
3. 希望: 「AIに任せる方法があります」
4. 具体: 「こうやって使います（実画面）」
5. 確信: 「やってみたら本当に30秒でした」
6. 行動: 「まずこれだけ試してみてください」
```

### 4. 社会的欲求

他者との関係における欲求を刺激する。

**このアカウントでの使い方:**

| 欲求 | OK（自然な訴求） | NG（煽り） |
|------|-----------------|-----------|
| 取り残されない | 「知っておくと来月楽になります」 | 「知らないと置いていかれます」 |
| 認められたい | 「上司に『それどうやったの？』と聞かれます」 | 「周りから一目置かれる存在に」 |
| 仲間意識 | 「同じ悩みの人、多いです」 | 「選ばれた人だけのコミュニティ」 |

**煽り・独占性・限定感の演出は使わない。** このアカウントの信頼を壊す。

### 5. ストーリーによる欲求喚起

成功物語で「自分もこうなれる」と思わせる。

**このアカウントでの使い方:**
- キャラの掛け合いがストーリーの役割を果たす
- ChatGPTやClaudeが「実際にやってみた」結果を語る
- 説得より物語のほうが抵抗が少ない

## 記事構成への適用

```
タイトル ──── 欲求トリガー（ペイン or 好奇心）
リード ────── コントラスト（Before → After の予告）
本文前半 ─── 共感 → 認識（感情エスカレーション 1-2）
本文中盤 ─── 希望 → 具体（感情エスカレーション 3-4）
図解 ──────── Before/Afterの可視化
掛け合い ──── ストーリーによる欲求喚起
本文後半 ─── 確信（実画面 or 具体結果）
CTA ──────── 行動（今日やれること1つ）
```
