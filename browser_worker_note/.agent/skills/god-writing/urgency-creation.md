# 行動喚起・緊急性創出（note / X 特化）

NOT Act（行動しない）を突破し、「あとでやろう」を「今やってみよう」に変えるスキル。

---

## 前提: このアカウントの緊急性

セールスページではないので、「残り3時間」「限定10名」は使えない。
代わりに使える緊急性を整理する。

| 一般的な緊急性 | 使えるか | このアカウントでの代替 |
|---------------|---------|---------------------|
| 時間的締切 | △ | 「ベータ版の今だから試せる」「日本上陸前に知っておく」 |
| 数量限定 | × | セールスではないので不適 |
| 機会損失 | **○** | **「知らないまま手作業を続ける時間がもったいない」** |
| 遅延コスト | **○** | **「毎月〇時間の手作業 = 知れば今日から減らせる」** |
| 変化の波 | **○** | **「周りが使い始めてから焦るより、今知っておく」** |

---

## このアカウントで使える3つの緊急性

### 1. 時間コストの可視化（最も効果的）

ミサキ層は「毎日やってるあの作業」に時間を使っている。その時間を数字で見せる。

```
OK: 「VLOOKUP調べる → 5分。月に20回 → 毎月100分。年間20時間。」
→ 身体感覚で「もったいない」と思える

OK: 「このアドインで数式作成が30秒。今まで5分かけてた作業が10分の1。」
→ Before/Afterが明確

NG: 「毎月〇〇円の機会損失です」
→ ミサキ層に響かない（給料制だから）
```

### 2. 周囲との差の意識

「みんなが使い始めてから焦る」より「今知っておく」方が楽、という文脈。

```
OK: 「チームの誰かがChatGPTでレポート作り始めたら、手作業の人は焦る」
→ 想像できるシーン

OK: 「隣の席の先輩がAIで30分で終わらせてるのを見てから調べるより、今知っておいたほうが楽」
→ ソフトな緊急性

NG: 「ライバルはすでに動いている！」
→ 煽り。驚き屋ムーブ

NG: 「AIを使えない人は淘汰される」
→ 恐怖煽り。NGトーンに抵触
```

### 3. タイミングの価値

「今この瞬間」に意味がある文脈。

```
OK: 「ベータ版の今なら無料で試せる。正式版では有料になるかも」
→ 事実に基づくタイミング

OK: 「日本上陸前に仕組みを知っておけば、来たときすぐ使える」
→ 準備の価値

NG: 「このチャンスを逃すと二度とありません」
→ セールスコピー。このアカウントに不適
```

---

## 使ってはいけない緊急性

このアカウントのポジショニング（驚き屋のアンチテーゼ）と矛盾する手法。

| NG手法 | 理由 |
|--------|------|
| 虚偽の締切 | 信頼崩壊。狼少年になる |
| 「残りわずか」 | セールスページではない |
| 恐怖煽り | 「AIに仕事を奪われる」= NGトーン |
| 煽り表現 | 「やばい、今すぐ始めないと」= 驚き屋 |
| カウントダウン | noteの記事にタイマーは不自然 |

**原則: 嘘の緊急性は使わない。事実に基づく「今知る価値」だけを伝える。**

## CTAでの緊急性の使い方

### パターン1: 時間コスト可視化 → 行動提案

```
「Excel関数を調べる時間、月に何時間使ってますか？
 ChatGPTに聞くだけで、その時間は今日からなくなります。
 まずはChatGPTを開いて「VLOOKUPの書き方教えて」と聞いてみてください。」
```

### パターン2: 準備の価値 → 情報提供

```
「ChatGPT for Excelは日本ではまだ使えません。
 でも仕組みを知っておけば、来たときにすぐ使えます。
 今日やれること: ChatGPTにExcelファイルを渡して分析させてみる。」
```

### パターン3: 小さな一歩 → 体験の誘導

```
「難しそうに見えますが、やることは1つだけ。
 ChatGPTを開いて、Excelの困りごとを日本語で話すだけ。
 5分で「こんなことできるのか」が分かります。」
```

## note記事での配置

```
リード ────── タイミングの価値（「〇〇が発表された今、知っておくこと」）
本文 ──────── 時間コストの可視化（Before/Afterの具体的な数字）
掛け合い ──── キャラが「やらない理由」を代弁 → 崩す
CTA ──────── 小さな一歩の提案（「まず〇〇だけやってみて」）
末尾 ──────── 次回予告で継続接点（「日本に来たら実画面レポートします」）
```

## X投稿での緊急性

140文字で緊急性を出すパターン:

```
パターンA: 時間コスト
「Excel関数、毎回ググってない？
 ChatGPT for Excelが来た。日本語で指示するだけ。
 日本上陸はまだだけど、今のChatGPTでも似たことはできる。」

パターンB: タイミング
「ChatGPT for Excel、ベータ版が3カ国で開始。
 日本はまだだけど、仕組みを知っておくと来たとき速い。
 できること・できないこと、まとめました。」
```

## チェックリスト

- [ ] 緊急性の理由は事実に基づいているか
- [ ] 身体感覚で理解できる数字を使っているか（時間・回数）
- [ ] 煽り表現（「やばい」「急いで」）を使っていないか
- [ ] 恐怖煽り（「取り残される」「淘汰される」）になっていないか
- [ ] CTAは「小さな一歩」を1つだけ示しているか
- [ ] 虚偽の締切・限定を使っていないか
- [ ] 価値と信頼を伝えた後に緊急性を配置しているか
