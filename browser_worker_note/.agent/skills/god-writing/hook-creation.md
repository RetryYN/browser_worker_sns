# フック作成（note / X 特化）

見出しの次に来る冒頭部分で、読者を本文に引き込む仕掛け。
NOT Read突破の第2関門（第1関門 = タイトル/見出し）。

---

## フックの役割

```
タイトルで止まる → フックで引き込む → 本文を読む
```

フックが弱いとタイトルで引いた読者が離脱する。

---

## このアカウントで使うフックタイプ

| タイプ | 効果 | 使いどころ |
|--------|------|-----------|
| 共感型 | 「自分のことだ」と思わせる | 最頻出。ミサキ層の「あるある」 |
| 事実型 | 信頼 + 好奇心 | ニュース記事のリード |
| 質問型 | 読者を参加させる | ペイン起点の記事 |
| 結果型 | 「やってみた」の結論先出し | 実演記事 |

### 使わないフックタイプ

| NGタイプ | 理由 |
|---------|------|
| 挑発型 | 「これを知らないと損」= 煽り。驚き屋 |
| ストーリー型 | 「3年前、私は〜」= このアカウントに個人の物語はない |
| 約束型 | 「この記事で人生が変わります」= 誇大。信頼崩壊 |

---

## フックのパターン（note記事リード）

### パターン1: 共感 → 結論（最頻出）

```
「VLOOKUP、毎回ググってませんか。
 ChatGPT for Excelが来ました。
 日本語で指示するだけで数式を作れるアドインです。
 ただし日本はまだ。今日できることも紹介します。」
```

構造: ペイン共感 → ニュース → ベネフィット → 制限 + 代替

### パターン2: 事実 → 影響（ニュース記事）

```
「3月5日、OpenAIがChatGPT for Excelを発表しました。
 Excelの中でAIに日本語で指示できるようになります。
 現在は米・加・豪のベータ版。日本対応は未定です。」
```

構造: 事実 → 読者への影響 → 条件

### パターン3: 質問 → 回答（ペイン起点）

```
「Excelで『この関数なんだっけ？』って毎回なりませんか。
 もうすぐ、その時間がなくなるかもしれません。」
```

構造: 読者のペインを質問で代弁 → 解決の予告

### パターン4: 結果 → 経緯（実演記事）

```
「ChatGPTにExcelファイルを渡して『売上の推移を分析して』と聞いたら、
 グラフ付きのレポートが30秒で出てきました。」
```

構造: やってみた結果 → 手順は本文で

## フックの長さ

| 媒体 | 長さ | 目安 |
|------|------|------|
| note記事リード | 3〜4行 | 全結論を含む（3NOT: リードに結論） |
| X投稿 | 1〜2行 | 140文字の冒頭。ここで止まらなければ終わり |
| 記事内H2直下 | 1〜2行 | セクションのトピックセンテンス |

---

## 共通ルール: リードに全結論を入れる

このアカウントの鉄則。結論を隠して引っ張る構成は使わない。

```
リードに含めるべき情報:
1. 何が起きたか（事実）
2. 何が嬉しいか（ベネフィット）
3. 何がダメか（制限・デメリット）
4. 今日やれること（代替 or 手順の予告）

NG:
「驚きの発表がありました。詳しくは本文で。」
→ 釣り。信頼崩壊
```

---

## フックから本文への接続

フックと本文は自然につなげる。唐突な切り替えはNG。

```
OK:
リード「〜ただし日本はまだ。今日できることも紹介します。」
→ H2「ChatGPT for Excelでできること」
→ 自然に本題へ

NG:
リード「〜今後が楽しみですね。」
→ H2「機能一覧」
→ リードと本文が断絶
```

---

## X投稿のフック

1行目がフック。ここで止まらなければ終わり。

```
OK:
「Excel関数、毎回ググってない？」
→ 共感型。スクロールが止まる

OK:
「ChatGPT for Excel来た。日本語で数式作れる。」
→ 事実型。ニュース + ベネフィット

NG:
「【速報】OpenAIが新機能を発表！」
→ 驚き屋フォーマット

NG:
「ChatGPT for Excelについてまとめました。」
→ フックがない。止まらない
```

---

## 掛け合いでのフック

記事冒頭に掛け合いを置く場合、キャラがフックの役割を果たす。

```
アイコ「Excel関数、毎回調べるのつらくない？」
ChatGPT「任せてくれたら5秒で終わるけど」
アイコ「…それExcelの中でやってくれるの？」
ChatGPT「やるよ。ChatGPT for Excel、出たから」
```

→ 読者の共感（アイコ） + ベネフィットの提示（ChatGPT）
