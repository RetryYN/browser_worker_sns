# タグライン・キャッチコピー作成（note / X 特化）

ブランドの本質を一言で表現するスキル。
プロフィール、固定ポスト、マガジンタイトルに使う。

---

## このアカウントのブランド本質

| 要素 | 内容 |
|------|------|
| ポジショニング | AI驚き屋のアンチテーゼ |
| 約束 | 一次情報を実画面つきで正直に見せる |
| トーン | 先輩が隣で見せてくれる感じ |
| 差別化 | 実画面スクショ + キャラ + 正直さ |

---

## 使いどころ

| 場所 | 文字数 | 目的 |
|------|--------|------|
| Xプロフィール | 5〜10文字 | 一言でアカウントを表現 |
| note固定記事タイトル | 10〜15文字 | 世界観の提示 |
| マガジンタイトル | 7〜13文字 | シリーズのテーマ |
| 記事内キャッチ | 7〜13文字 | セクションの引き |

---

## 作成プロセス

1. ブランド本質のキーワードを抽出
2. **20案以上出す**（量が質を生む）
3. 声に出してリズム確認
4. ミサキ層が「自分向けだ」と感じるか確認
5. 絞り込み

---

## このアカウントで使うタイプ

| タイプ | 例 | 効果 |
|--------|-----|------|
| 宣言型 | 「使ってみた、が全て。」 | スタンスの明示 |
| ベネフィット型 | 「AIの隣の席から。」 | トーンの体現 |
| 対比型 | 「煽らない。見せる。」 | 驚き屋との差別化 |

### 使わないタイプ

| NGタイプ | 理由 |
|---------|------|
| 煽り型 | 「AI革命、始まる。」= 驚き屋 |
| 命令型 | 「今すぐAIを使え」= 上から目線 |
| 抽象型 | 「可能性は無限大」= 何のアカウントかわからない |

---

## テクニック

| テクニック | 例 |
|-----------|-----|
| 対比 | 「驚くな。試せ。」 |
| 体言止め | 「実画面の説得力。」 |
| 反復 | 「触って、見せて、正直に。」 |
| 韻 | リズムで記憶に残す |

---

## 良いタグラインの条件

- [ ] 短いか（5〜13文字）
- [ ] 声に出しやすいか
- [ ] このアカウントだけのものか（他のAI系アカウントに使えないか）
- [ ] ミサキ層に響くか
- [ ] 驚き屋と差別化できているか
- [ ] 20案以上の中から選んだか
