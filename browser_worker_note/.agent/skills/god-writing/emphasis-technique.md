# 強調テクニック（note / X 特化）

重要な情報を際立たせ、読者の注意を引く技法。
「全てを強調すると、何も強調されない」が大前提。

---

## 強調の優先度

| レベル | 箇所数 | このアカウントでの使いどころ |
|--------|--------|---------------------------|
| 最重要 | 記事で1〜2箇所 | 「今日やれること」「日本では使えない」等の核心 |
| 重要 | セクションごと1箇所 | 各見出しのキーメッセージ |
| 補足 | 必要最小限 | 数字・固有名詞の強調 |

---

## 位置による強調

記憶に残りやすい位置: **文頭・文末、段落冒頭・末尾、記事冒頭・結末**

| 原理 | 効果 | このアカウントでの使い方 |
|------|------|----------------------|
| 初頭効果 | 最初の情報が印象に残る | リードに全結論を入れる（3NOT突破と連動） |
| 親近効果 | 最後の情報が印象に残る | CTAを記事末尾に配置 |

```
OK:
「日本ではまだ使えません。」← 文頭に制限を置く = 誠実さの強調
「まずChatGPTにExcelファイルを渡してみてください。」← 末尾にCTA = 行動の強調

NG:
「〜という機能があり、ただし日本ではまだ使えないのですが、将来的には〜」
→ 制限が文中に埋もれる
```

## 対比による強調

反対のものと並べて際立たせる。このアカウントの武器。

### Before/After（最頻出）

```
「VLOOKUP調べて手入力: 5分。ChatGPTに聞く: 30秒。」
→ 数字の対比で変化が一目でわかる
```

### 驚き屋 vs このアカウント

```
「驚き屋: 『やばい！神アプデ！！』
 このアカウント: 実画面スクショ + 制限事項。」
→ 対比でポジショニングを明確化（直接使う場面は少ないが、設計思想として）
```

### キャラの対比

```
ChatGPT「Excel？ 任せて。」
Claude「私もファイル分析は得意ですよ。」
→ キャラの個性が対比で際立つ
```

## 短文による強調

長文の流れの中に短い一文を挟むとインパクトが出る。

```
OK:
「ChatGPT for Excelは、Excelの中で自然言語で指示できるアドインです。
 数式の作成、エラー修正、シナリオ分析ができます。
 ただし日本はまだ。」
→ 最後の短文が強く印象に残る

OK:
「精度87%。前世代から大幅向上。」
→ 体言止め + 短文の連続でリズムと強調

NG:
「衝撃。」「革命。」「神。」
→ 驚き屋ムーブ。このアカウントでは使わない
```

**多用すると効果が薄れる。** 1セクションに1回が目安。

---

## 反復による強調

重要なキーワードを意図的に繰り返す。3回程度が上限。

```
OK:
「日本語で指示するだけ。
 関数を知らなくても、日本語で指示するだけ。
 それがChatGPT for Excelの核心です。」
→ 「日本語で指示するだけ」の反復で核心を刷り込む

NG:
「すごい。すごい機能。本当にすごい。」
→ 中身のない反復。驚き屋と同じ
```

---

## 視覚的強調（noteエディタで使えるもの）

| 手法 | 使いどころ | 注意 |
|------|-----------|------|
| **太字** | 本当に重要な箇所のみ。全体の10-20% | 太字だらけ = 太字なし |
| 見出し（H2/H3） | セクション区切り | スキャンで内容が把握できる設計 |
| 箇条書き | 3項目以上の並列情報 | 項目は短く |
| 表 | 比較・スペック | 3列以内（モバイル対応） |
| 引用ブロック | 公式発表の引用 | 引用元を明記 |
| 図解・画像 | 情報の区切り + 視覚的休息 | 300文字ルールと連動 |

### noteでのNG

- 下線: リンクと混同する
- 色文字: noteエディタの標準機能外
- 文字サイズ変更: noteでは見出し以外で不可

## 記号による強調

| 記号 | 使い方 | このアカウントでの扱い |
|------|--------|---------------------|
| ？ | 問いかけ | 見出しで使う（「毎回ググってない？」） |
| 。 | 体言止めの区切り | 短文強調に有効（「精度87%。」） |
| — | 挿入・補足 | 補足情報の挿入（「ただし — 日本はまだ」） |
| … | 余韻 | 掛け合いで使う（アイコ「…一人もまともに答えてない」） |
| ！ | 強い感情 | **掛け合いのみ**。情報パートでは使わない |

**情報パートで「！」を多用すると驚き屋っぽくなる。** 掛け合い・キャラのセリフに限定。

---

## 組み合わせの例

効果的: 2〜3種類の組み合わせ。

```
位置（文頭） + 短文 + 太字:
「**日本ではまだ使えません。**」

対比 + 数字 + 表:
| Before | After |
| VLOOKUP調べて5分 | ChatGPTに聞いて30秒 |

反復 + 位置（末尾）:
「まず聞いてみてください。日本語で聞くだけです。」
```
