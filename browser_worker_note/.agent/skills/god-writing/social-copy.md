# SNSコピーライティング（note / X 特化）

プラットフォーム特性に合わせたコピー作成スキル。
このアカウントが使うのはnoteとXのみ。

---

## 原則

| 原則 | このアカウントでの実践 |
|------|---------------------|
| 短く | スクロール中に読まれる長さ |
| 会話的 | 先輩が隣で話す感じ |
| 価値提供 | シェアしたくなる情報 |
| 視覚連携 | 図解・キャラ画像と組み合わせ |
| エンゲージメント誘発 | リプ・コメントを促す |
| ネイティブ感 | 宣伝っぽくない |

---

## X投稿のコピー設計

### 構造

```
1行目: フック（止まらせる）
本文: 価値（140文字で完結）
末尾: CTA or 次への誘導
画像: 図解 or キャラ（テキストだけの投稿は避ける）
```

### 1行目のパターン

| パターン | 例 |
|---------|-----|
| ペイン共感 | 「Excel関数、毎回ググってない？」 |
| ニュース | 「ChatGPT for Excel来た。」 |
| 結果先出し | 「ChatGPTにExcel渡したら30秒でレポート出てきた」 |
| 問いかけ | 「Excelで一番時間使ってる作業、何？」 |

### エンゲージメント獲得

| 手法 | 例 |
|------|-----|
| 質問 | 「あなたのExcelあるある教えて」 |
| 選択肢 | 「VLOOKUP派？ INDEX MATCH派？」 |
| 体験共有 | 「ChatGPTに聞いた数式で一番助かったのは？」 |

### ハッシュタグ

X: 2〜3個。投稿末尾に配置。

```
#ChatGPT #Excel #AI業務効率化
```

多すぎるとスパム感。少なく絞る。

## note記事のコピー設計

### タイトル（28〜32文字）

→ headline-writing.md 参照

### リード（3〜4行）

→ hook-creation.md 参照

### 記事内の文章トーン

| パート | トーン |
|--------|--------|
| 情報パート | 落ち着いた説明調。「〜です。〜ます。」 |
| 掛け合い | キャラの個性全開。口癖・ツッコミ |
| CTA | 親しみ + 具体的。「まず〇〇してみてください」 |

### CTA文のパターン

```
行動促進: 「ChatGPTを開いて『VLOOKUPの書き方教えて』と聞いてみてください。」
期待形成: 「日本に来たら実画面つきでレポートします。」
参加誘導: 「試してほしい業務があればコメントで教えてください。」
```

## 広告っぽくならないためのルール

| NG | OK |
|----|-----|
| 「ぜひフォローしてください！」 | 自然に価値を出せばフォローされる |
| 「いいねお願いします」 | 問いかけでエンゲージメントを誘発 |
| 「〇〇のまとめ記事はこちら」 | 記事の内容で引く |
| 絵文字の乱用 | 必要最小限 |
