# ビジネスライティング スキル
**対象領域: note/X コンテンツ投稿**

note/Xでのコンテンツ運営における各種ビジネスコミュニケーション文書。コラボ依頼、読者への連絡、運営報告など。

## Use this skill when
- コラボ・寄稿依頼のメールやDMを作成するとき
- 読者・フォロワーへの連絡文（お知らせ・お礼等）を作成するとき
- 運営報告・活動レポートなどの共有文書を作成するとき
- ビジネスメールのトーン・マナーを整えるとき

## Do not use this skill when
- セールス目的の販売ページ・販促文を求められたとき（-> sales-writing）
- データ分析や論理的な報告書を求められたとき（-> logical-writing）

## 🤖 System Prompt（AIへの指示前提）
```text
あなたはnote/Xでコンテンツ発信を行うクリエイターの優秀なビジネスパートナーです。
論理的でありながら、人を不快にさせない温かみとプロフェッショナルさを兼ね備えたビジネスメール・文書を作成します。
```

## 基本原則

### 1. 結論ファースト（PREP法）
- **Point**: 結論を最初に
- **Reason**: 理由を述べる
- **Example**: 具体例で補強
- **Point**: 結論を再度

### 2. 簡潔性
- 1文60文字以内
- 冗長表現の排除（「〜させていただく」→「〜します」）
- 箇条書きの活用

### 3. 相手別の温度感
- コンテンツ発信は「人」との繋がりが命。冷たい文面は致命的
- 事務的すぎず、馴れ馴れしすぎないバランス

## 文書タイプ別テンプレート

### コラボ・寄稿依頼メール
```
件名: 【ご相談】〇〇テーマでのコラボ記事について

〇〇様

はじめまして。[名前]と申します。
note/Xで〇〇に関する発信をしています。

■ ご連絡の目的
〇〇テーマでのコラボ記事を企画しており、
〇〇様にぜひご参加いただきたくご連絡しました。

■ 企画概要
1. テーマ: [具体的に]
2. 形式: [対談 / 寄稿 / 共同執筆]
3. 公開予定: [時期]
4. 読者層: [ターゲット]

■ 〇〇様にお願いしたい理由
[具体的な発信内容への言及・リスペクト]

ご興味がありましたら、一度オンラインでお話しできればと思います。
お忙しいところ恐れ入りますが、ご検討いただけますと幸いです。
```

### 読者へのお知らせ（noteの更新通知等）
```
いつも記事を読んでくださり、ありがとうございます。
[名前]です。

■ お知らせ
[内容を簡潔に]

■ 背景・理由
[なぜこの変更/お知らせをするのか]

■ 今後の予定
[次のアクション]

引き続き、皆さまに役立つ情報をお届けしていきます。
ご質問やご要望があれば、お気軽にコメントやDMでお知らせください。
```

### 活動報告・月次レポート
```
■ 期間: [YYYY年MM月]
■ 投稿数: [X]本
■ 総PV: [X]
■ フォロワー増減: [+X]

  1. [記事タイトル]（[PV]）

  3. [記事タイトル]（[PV]）

  - [うまくいったこと]
```

### お礼メッセージ（フォロワー・読者向け）
```
[名前]様

コメント（/ シェア / ご購入）いただき、ありがとうございます。
[具体的な内容への言及]

とても励みになります。
今後も〇〇をテーマに発信を続けていきますので、
引き続きよろしくお願いいたします。
```

## トーン＆マナー（note/Xコンテンツ運営）

| 相手 | トーン | 注意点 |
|---|---|---|
| コラボ相手（クリエイター） | リスペクト＋提案型 | 相手の発信内容に具体的に触れる |
| 読者（フォロワー） | 温かく＋プロフェッショナル | 上から目線NG、感謝を忘れない |
| 新規読者（初コメント） | 歓迎＋丁寧 | テンプレ感を消す |
| 編集者・メディア | ビジネスライク＋簡潔 | 事実ベース、実績を添える |

## NG表現

| NG | OK | 理由 |
|---|---|---|
| 「フォロワーを集める」 | 「読者と繋がる」 | 人を数字扱いしない |
| 「まだ読んでませんか？」 | 「よろしければご覧ください」 | 圧をかけない |
| 「バズった」 | 「多くの方に読まれた」 | ビジネス文書では砕けすぎ |
| 「見てください」 | 「ご覧いただけますと幸いです」 | 丁寧さを保つ |
