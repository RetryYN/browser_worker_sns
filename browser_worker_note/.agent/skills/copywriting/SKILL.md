# コピーライティング スキル
**対象領域: note/X コンテンツ投稿**

形のないアイデアや価値を、短い言葉で可視化する。読者の心を掴み、クリック・シェア・フォローなどの行動を促すコピーの作成技法。

## Use this skill when
- キャッチコピー・タグラインの作成を求められたとき
- X投稿用の短文コピーが必要なとき
- noteの記事タイトル（刺さる見出し）を求められたとき
- LP（ランディングページ）の見出しを求められたとき

## Do not use this skill when
- 長文のストーリー記事を求められたとき（-> storytelling）
- BtoB向けの提案書やメールを求められたとき（-> sales-writing）

## 🤖 System Prompt（AIへの指示前提）
```text
あなたはnote/X投稿に特化したトップクラスのコピーライターです。
抽象的な魅力を、読者の心が動く「具体的で短い言葉」に変換するスキルを持っています。
出力するコピーは最大でも13文字以内、最強は7文字であるという鉄則を遵守してください。
```

## コピーの2大分類

### イメージコピー（ブランディング目的）
- **目的**: 発信者の世界観・価値観を伝える。直接売らない
- **効果**: 記憶に残る、好感度を上げる、「この人の発信をもっと読みたい」を醸成
- **note/Xでの使いどころ**: プロフィール、固定ポスト、マガジンタイトル

```
「書く、を変える。」（7文字）
「あなたの"好き"が記事になる。」（13文字）
「言葉が、未来をつくる。」（10文字）
「出会いを、チカラに。」（9文字）
「可能性は、無限大。」（8文字）
```

### セールスコピー（クリック・登録喚起）
- **目的**: 今すぐクリック・フォロー・シェアさせる
- **効果**: PV数、フォロワー数、エンゲージメント率直結
- **note/Xでの使いどころ**: 記事タイトル、ポスト冒頭、CTA文

```
「3日で1000PV、達成。」
「フォロワー0から始めた。」
「まず読んでみて。」
```

## 文字数の鉄則

> **最強は7文字。最大でも13文字。**

| 文字数 | 評価 | 理由 |
|---|---|---|
| **7文字** | ★★★ 最強 | 一目で入る、記憶に残る |
| 8〜10文字 | ★★☆ 良い | 十分短い |
| 11〜13文字 | ★☆☆ 許容 | これが限界 |
| 14文字以上 | ☆☆☆ NG | 削り直す |

## 無形コンテンツ・コピーの最大の壁

> **「形がない = 想像できない = 価値がわからない」**

### 解決策: 抽象を具体に変換する

| 無形のまま（NG） | 具体化（OK） |
|---|---|
| 「役に立つ情報」 | 「売上、2倍になった。」 |
| 「成長できるヒント」 | 「読んだ翌日に結果出た。」 |
| 「充実のノウハウ」 | 「3分で即実践。」 |
| 「読みやすい記事」 | 「電車1駅で読了。」 |
| 「心に響くコンテンツ」 | 「"ありがとう"が日常。」 |

## コピー作成フレームワーク

### 型1: 体言止め（最短・最強）
```
「次の一手。」（5文字）
「書く幸せ。」（5文字）
「本気の発信。」（6文字）
```

### 型2: Before → After（変化を見せる）
コンテンツは「変化」が最大の価値。
```
「迷いが、確信に変わった。」（11文字）
「"嫌い"が"好き"に変わる。」（11文字）
「月曜が怖くなくなった。」（11文字）
```

### 型3: 読者の本音を代弁
```
「もう我慢しなくていい。」（11文字）
「このままでいいの？」（9文字）
「好きなことで、生きる。」（10文字）
```

### 型4: 数字+ベネフィット
note/Xで刺さる数字 = PV、フォロワー、収益、時間
```
「月1万PVの裏側。」（9文字）
「週1投稿で収益化。」（9文字）
「フォロワー1万到達。」（10文字）
```

### 型5: 呼びかけ型（ターゲット直撃）
```
「30代、本気の発信へ。」（10文字）
「初心者のあなたへ。」（9文字）
「書くのが怖い人へ。」（9文字）
```

### 型6: 断言・宣言型
```
「人生は、変えられる。」（9文字）
「発信する時代。」（7文字）
「妥協しない。」（6文字）
```

## ブラッシュアップ手法

### Step1: 書き出す（20案以上、質より量）
### Step2: 13文字以下に削る
```
Before: 「あなたの発信を支え最適な情報をお届けします」（21文字）
After:  「寄り添う言葉。」（7文字）
```

### Step3: "形のない価値"を五感に変換する
```
Before: 「やりがいのある発信」
After:  「鳥肌が立つ言葉。」（8文字）
```

### Step4: 音読してリズム確認
- 七五調で整える
- 句読点「、」「。」でリズムを作る

## コピー評価チェックリスト

- [ ] **13文字以内か？**（7文字なら最高）
- [ ] **無形の価値が"見える化"されているか？**
- [ ] **読者が"自分ごと"と感じるか？**
- [ ] **声に出して気持ちいいリズムか？**
- [ ] **イメージ or セールス、目的に合っているか？**
- [ ] **他の発信者と差別化できているか？**

## 媒体別最適化ガイド（note/X投稿）

| 媒体 | 文字数 | 構成ポイント | エンゲージメント向上の鍵 |
|---|---|---|---|
| X（ポスト） | 7〜10文字+絵文字 | 共感・本音代弁型が強い | リプ・引用RTで拡散→プロフィール誘導 |
| X（プロフィール） | 7文字 | 発信者の本質を一言で | 全媒体で統一使用 |
| note記事タイトル | 7〜13文字 | 数字+ベネフィット | 検索結果・おすすめ一覧で目立つこと |
| noteマガジン名 | 7〜10文字 | 世界観・テーマ性 | シリーズとしての統一感 |
| Instagram/Reels | 7文字（テキスト少なめ） | ビジュアル主体、テキストは補助 | プロフィールリンクへの動線 |
| LP見出し | 7〜13文字 | 痛みor憧れ、スクロール誘導 | ファーストビューで離脱を防ぐ |

### 媒体別の注意点
- **X**: 7文字ルールが最も活きる。短さ=スクロール停止力
- **note**: 記事タイトルは検索流入も意識。キーワードを含めつつ短く
- **Instagram**: ビジュアルが主役。コピーはあくまで補助
