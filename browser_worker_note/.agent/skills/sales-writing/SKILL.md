# セールスライティング スキル
**対象領域: note/X コンテンツ投稿**

コンテンツやサービスの価値を伝え、読者の行動（購入・登録・フォロー）を促す文章作成技法。

## Use this skill when
- 有料noteやサービスの販売ページ・紹介文を作成するとき
- サービスLPの本文構成を求められたとき
- フォロワー獲得・メルマガ登録を促すCTA文を作成するとき
- コンテンツの販促用ポスト・紹介文を作成するとき

## Do not use this skill when
- 社内メールや事務連絡を求められたとき（-> business-writing）
- キャッチコピーだけを求められたとき（-> copywriting）
- データ中心のレポートを求められたとき（-> logical-writing）

## 🤖 System Prompt（AIへの指示前提）
```text
あなたはnote/Xでのコンテンツ販売・集客に精通したトップセールスライターです。
有形商材と異なり「実績と信頼」がすべてである無形商材の特性を理解し、
顧客の課題（痛み）を指摘しつつ、論理と感情の両面から行動（購入・登録等）を促すコピーを作成します。
```

## 無形商材セールスの最大の課題

> **見えない、触れない、試せない → 信頼がすべて**

### 有形 vs 無形の違い

| | 有形商材 | 無形商材（コンテンツ） |
|---|---|---|
| 購入前 | 実物を見れる | 見えない |
| 判断基準 | スペック比較 | 信頼・実績 |
| 効果実感 | 即時 | 時間がかかる |
| 失敗リスク | 返品可能 | 時間・お金の損失 |

→ **だから「信頼」と「具体性」で売る**

## セールスフォーミュラ

### PAS（Problem → Agitate → Solve）
```markdown
【Problem: 読者の痛みを言語化】
「毎日投稿しているのに、フォロワーが全然増えない経験はありませんか？」

【Agitate: 痛みを掘り下げる】
「時間をかけて書いた記事が読まれない。
 "いいね"はつかず、コメントもゼロ...
 "自分には才能がないのか"と思い始めていませんか？」

【Solve: 解決策を提示】
「[コンテンツ名]は、フォロワー0から始めた筆者が
 6ヶ月で1万フォロワーを達成した具体的な手法をまとめています。」
```

### BAB（Before → After → Bridge）
```markdown
【Before: 読者の現状】
「記事を書いても読まれない。SNSで発信しても反応がない。
 何が間違っているのかわからず、モチベーションが下がる日々...」

【After: 理想の未来】
「投稿するたびに通知が鳴る。記事が拡散され、新しい読者がやってくる。
 "あなたの記事で救われました"というコメントが届く。」

【Bridge: コンテンツが橋渡し】
「[コンテンツ名]が、ゼロから読まれる発信者になるまでのロードマップを提供します。」
```

## 販売ページ・紹介文テンプレート

### 有料noteの販売ページ
```markdown
## 1. フック: 読者の悩みに共感するキャッチ
「なぜ、あなたの記事は読まれないのか？」

## 2. 共感: 自分も同じだったストーリー
「実は私も最初の半年間、PVは1日10以下でした...」

## 3. 転換: 気づきと解決策の提示
「ある日、たった1つの書き方を変えただけで...」

## 4. コンテンツ内容: 何が学べるか（目次形式）
- Chapter 1: ...
- Chapter 2: ...

## 5. 実績・読者の声
「この方法で月間1万PVを達成しました」（読者Aさん）

## 6. 価格と保証
「通常価格 ¥X,XXX → 今だけ ¥X,XXX」

## 7. CTA
「今すぐ読んで、明日の投稿から変わる」
```

### X販促ポストの構成
```markdown
【フック（1行目で止める）】
「フォロワー0→1万を達成した"たった3つの習慣"」

【価値の提示（箇条書き）】
✅ 毎日投稿より大事な○○
✅ プロフィールの△△を変えるだけ
✅ □□の時間帯に投稿する理由

【CTA】
「詳しくはnoteにまとめました👇」
```

## LP構成（コンテンツ・サービス向け）

### 対読者（toC）
```
1. ヘッド: 読者の悩みに共感するキャッチ
2. 共感: 「こんな悩みありませんか？」3つ
3. サービス紹介: 何が得られるのか（図解）
4. 実績: 利用者の声（実名+数字）
5. 流れ: 購入→実践→成果（ステップ図）
6. Q＆A: 「初心者でも大丈夫？」「すぐに効果出る？」
7. CTA: 「今すぐ始める」
```

## 信頼構築パーツ（無形商材の生命線）

| パーツ | note/Xでの例 |
|---|---|
| 数字実績 | 「累計10万PV、フォロワー1万人」 |
| 読者の声 | 購入者・フォロワーのリアルな感想を掲載 |
| Before/After | 「月間100PV→月間1万PVに成長」 |
| リスクリバーサル | 「満足いただけなければ全額返金」 |
| プロセス開示 | 実際のやり方・手順をオープンに（ブラックボックスにしない） |
